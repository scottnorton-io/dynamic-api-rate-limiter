# Code of Conduct

This project is intended to be a safe, welcoming, and professional space for
everyone, regardless of background or experience level.

- Be respectful and considerate in all interactions.
- Assume good intent, but be open to feedback.
- Avoid personal attacks, harassment, and discriminatory language.
- Focus on technical issues and project improvements.

If you experience or witness unacceptable behavior in the context of this
project, please report it via one of:

- Opening an issue on GitHub (if you feel comfortable doing so), or
- Contacting the maintainer directly at: ideas@scottnorton.io

The maintainer reserves the right to take appropriate action, which may
include warnings, issue locking, or banning individuals from participation
in this project and its community spaces.
